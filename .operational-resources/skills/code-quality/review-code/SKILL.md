# Skill: Review Code

## Goal

Review có cấu trúc: correctness, design, security, tests, performance.

## Steps

1. Đọc diff + context (task + spec).
2. Correctness: logic, edge cases, concurrency, transaction.
3. Design: single responsibility, naming, coupling.
4. Security: input validation, authz, secret, logging.
5. Tests: coverage nhánh quan trọng.
6. Ghi comment actionable; phân loại must-fix vs nit.
