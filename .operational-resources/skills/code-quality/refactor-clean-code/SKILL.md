# Skill: Refactor Clean Code

## Goal

Refactor an toàn: cùng hành vi observable, test hồi quy, bước nhỏ.

## Steps

1. Viết hoặc củng cố test cho hành vi hiện tại.
2. Giải thích lại logic hiện tại (comment ngắn trong task file nếu cần).
3. Tách method/class; đổi tên rõ nghĩa; giảm lồng ghép.
4. Chạy full test suite liên quan module.
5. Không “cleanup” không liên quan trong cùng commit.
