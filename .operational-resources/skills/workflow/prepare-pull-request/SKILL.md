# Skill: Prepare Pull Request

## Goal

PR sạch: diff nhỏ, mô tả rõ, test plan, risk.

## Steps

1. `git diff` review: không file lạ, không format noise không cần.
2. Chạy test + lint/format theo team.
3. Mô tả PR: what, why, how to test, rollback.
4. Link ticket/issue; screenshot nếu UI.
5. Self-review checklist `rules/08-review-checklist.md`.
