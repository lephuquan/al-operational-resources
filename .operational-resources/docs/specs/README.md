# Feature Specifications (Specs)

Đây là thư mục chứa **đặc tả chi tiết từng tính năng** từ góc nhìn cá nhân.

## Mục đích

- Cung cấp yêu cầu business rõ ràng cho AI.
- Tham chiếu khi implement hoặc test tính năng.
- Đảm bảo nhất quán giữa code và yêu cầu.

## Cách dùng khi prompt Cursor

- "Refer to `specs/feature-order.md` for requirements"
- "Implement according to `specs/feature-auth.md`"

## Quy tắc

Mỗi tính năng lớn nên có một file spec riêng. Tạo mới từ `template.md`.
