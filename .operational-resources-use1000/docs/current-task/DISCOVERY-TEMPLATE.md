## 8) Handoff into task contract

Copy confirmed outputs into:

- `docs/current-task/YYYYMMDD-slug.md` section 2 (scope/assumptions)
- section 8 (context pack)
- section 10 (MUST/SHOULD/ASK FIRST)
- section 11 (open questions)

Then run:

```powershell
powershell -File .operational-resources/scripts/start-task.ps1 -TaskFile ".operational-resources/docs/current-task/YYYYMMDD-slug.md"
```
