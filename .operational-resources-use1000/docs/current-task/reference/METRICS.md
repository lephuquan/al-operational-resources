# Task Workflow Metrics

Track weekly metrics to measure AL workflow quality.

## Core KPIs

| KPI | Target | How to measure |
|-----|--------|----------------|
| First-pass AC pass rate | >= 80% | % tasks accepted without scope rework |
| Review rework rate | <= 25% | % tasks requiring major rework after review |
| Test evidence completeness | 100% | % merged tasks with test evidence |
| Task traceability completeness | 100% | % tasks with task file + log + PR link |
| Mean cycle time | Down trend | Start-task to done |

## Weekly snapshot template

| Week | Tasks done | AC pass rate | Rework rate | Traceability | Notes |
|------|------------|--------------|-------------|--------------|-------|
| YYYY-WW | 0 | 0% | 0% | 0% | |

## Usage

- Update once per week in review meeting.
- If any KPI misses target for 2 weeks, create a corrective action task.
