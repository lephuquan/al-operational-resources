# `reference/` — tài liệu vận hành quanh `current-task`

| File | Mục đích |
|------|----------|
| `FOLDER-CONVENTION.md` | Một work item = một thư mục; `TASK.md` / `DISCOVERY.md` |
| `WORKFLOW-4-PHASES.md` | Ánh xạ 4 phase automation + human |
| `MIGRATION-TO-STRICT-CONTRACT.md` | Nâng task legacy lên contract nghiêm |
| `METRICS.md` | KPI theo dõi workflow |
| `HUMAN-GATE-CHECKLIST.md` | Checklist sau handoff (Human) |
