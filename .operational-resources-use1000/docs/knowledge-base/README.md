# Knowledge Base (KB)

Kho tri thức, quy tắc và kinh nghiệm thực chiến để tối ưu cộng tác **Người + AI**.

## Mục tiêu

- Cung cấp context “cứng” để AI không lệch tiêu chuẩn dự án.
- Lưu giải pháp cho vấn đề lặp lại.
- Tinh chỉnh “Vibe Coding” với tốc độ cao nhưng vẫn an toàn.

## Danh mục

1. **Prompt Engineering**: `prompt-engineering-rules.md`
2. **Naming**: `naming-conventions.md`
3. **Patterns**: `coding-patterns.md`
4. **Security & privacy**: `security-privacy-rules.md`
5. **Troubleshooting**: `troubleshooting-manual.md`
6. **Error codes** (tham chiếu): `error-codes.md`

Tạo mục mới từ `template.md`.
